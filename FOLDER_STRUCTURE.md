## 🎯 Key Files to Configure

### 1. **src/config/constants.js** (REQUIRED)
Update with your Google Apps Script URL after deployment.

### 2. **public/icon-192.png & icon-512.png** (REQUIRED for PWA)
Create app icons or use a placeholder generator:
- [favicon.io](https://favicon.io)
- [realfavicongenerator.net](https://realfavicongenerator.net)

### 3. **src/data/workoutTemplate.js** (OPTIONAL)
Customize exercises, sets, and reps to match your workout plan.

### 4. **GOOGLE_APPS_SCRIPT.js** (REQUIRED)
Copy to Google Apps Script editor and update SPREADSHEET_ID.

## 📦 Component Architecture

### Screen Components (Route-level)
- `Home.jsx` - Landing screen
- `WorkoutLogger.jsx` - Workout input form
- `History.jsx` - Past workout viewer

### Reusable Components
- `ExerciseCard.jsx` - Input card for one exercise
- `SubmitButton.jsx` - Animated submit button with states

### Utilities
- `api.js` - External API communication
- `localStorage.js` - Browser persistence
- `dateHelpers.js` - Date calculations

### Configuration
- `constants.js` - App-wide config (⚙️ UPDATE THIS)
- `workoutTemplate.js` - Exercise definitions

## 🔄 Data Flow

```
User Input → WorkoutLogger Component
           ↓
      Validation (api.js)
           ↓
      localStorage backup
           ↓
      POST to Google Apps Script
           ↓
      Google Sheet Update
           ↓
      Success Confirmation
```

## 🎨 Styling Architecture

- **Tailwind CSS**: Utility-first styling
- **Custom CSS Classes**: `.glass`, `.glass-light` (index.css)
- **Framer Motion**: Animations and transitions
- **Lucide Icons**: Icon library

## 📱 PWA Files (Auto-generated by Vite PWA)

These are created automatically during build:
- `dist/sw.js` - Service worker
- `dist/manifest.webmanifest` - PWA manifest
- `dist/workbox-*.js` - Workbox cache strategies

## 🚀 Build Process

1. **Development**: `npm run dev`
   - Vite dev server
   - Hot module replacement
   - No PWA features

2. **Production**: `npm run build`
   - Optimized bundle
   - PWA manifest generated
   - Service worker registered
   - Output in `dist/` folder

3. **Preview**: `npm run preview`
   - Test production build locally
   - PWA features enabled
